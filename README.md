# 听云sourcemap-webpack-plugin使用demo

配合webpack插件引入文档使用

## webpack 5


```
cd webpack5
```

```
npm install
```

```
npm run build
```

打包生成的文件在`webpack5/dist`

## vue-cli

```
cd vue-cli
```

```
npm install
```

```
npm run build
```
打包生成的文件在`vue-cli/dist`

## create-react-app

```
cd create-react-app
```

```
npm install
```

```
npm run build
```

打包生成的文件在`create-react-app/dist`

## angular-cli


```
cd angular-cli
```

```
npm install
```

```
npm run build
```

打包生成的文件在`angular-cli/dist`
